## 파이썬 설치 가이드(Python 3.11.9)

### 파이썬 다운로드(Python 3.11.9)

* [파이썬 공식 홈페이지 다운로드 페이지](https://www.python.org/downloads/windows/)에 접속합니다.
* 목록에서 **Python 3.11.9**를 찾습니다. (`Ctrl` + `F`를 누르면 쉽습니다.)
* Windows용 설치 파일(Installer)을 다운로드합니다. **(일반적으로 64-bit 버전을 선택)**
* 다운로드된 파일을 실행합니다.

<img src=https://i.postimg.cc/BZpJXtcJ/image.png, width=600>

---

### 설치 시 주의사항 (핵심!)

설치 파일을 실행하면 나타나는 첫 화면이 가장 중요합니다.

<img src=https://i.postimg.cc/R0WTn3nN/image.png, width=600>

- **[필수 선택] Add Python 3.11 to PATH**: 이 체크박스를 반드시 선택해야 합니다.
    - **이유:** 체크하지 않으면 터미널(CMD)에서 `python` 명령어를 입력해도 컴퓨터가 파이썬을 찾지 못합니다.
 
---

### 설치 완료 및 확인

설치가 끝나면 `Setup was successful`이라는 문구가 뜹니다. 이제 설치가 잘 되었는지 검증해야 합니다.

#### **0. CMD창 띄우는 법**
1. 키보드의 **[Windows 로고 키 + R]**을 누른 뒤, 실행 창에 `cmd`를 입력하고 엔터를 누릅니다.
2. 또는 작업표시줄 검색창에 **'CMD'** 혹은 **'PowerShell'**을 검색하여 실행합니다.
3. **예시:**
<img src=https://i.postimg.cc/bJ3bCpbP/image.png, width=600>

#### **1. 0번에 따라 명령 프롬프트(CMD) 또는 PowerShell을 새로 엽니다. (이미 열려있던 창은 닫아야 합니다.)**

#### 2. 아래 명령어를 입력합니다.

```bash
python --version
```

## 주피터 노트북 설치 가이드

### 1. CMD 실행 및 pip 업데이트
* Windows 검색창에 **'CMD'**를 입력하여 실행합니다.
* **주의:** 파이썬 설치 직후라면, 기존 창을 닫고 **새 창**을 열어야 설치된 파이썬을 인식합니다.
* 아래 코드를 실행하여 pip을 업데이트 합니다.

```bash
python -m pip install --upgrade pip
```

---

### 2. 주피터 노트북 설치 명령어

터미널에 아래 명령어를 입력하고 엔터를 누릅니다. 우리가 정한 안정 버전인 **7.1.3** 버전을 설치합니다.

```bash
pip install notebook==7.1.3
```

---

### 3. 설치 확인

설치가 완료되었다면 아래 명령어를 입력하여 정상적으로 설치되었는지 버전을 확인합니다.

```bash
jupyter notebook --version
```
화면에 7.1.3이라는 숫자가 나오면 성공입니다.

---

### 4. 주피터 노트북 실행하기

이제 실제로 프로그램을 실행해 봅니다. 터미널(CMD)에 다음 명령어를 입력하세요.

```bash
jupyter notebook
```

결과: 잠시 후 검은색 터미널 창에 로그 기록이 올라오면서, 자동으로 웹 브라우저(Chrome, Edge 등)가 실행되고 주피터 노트북 대시보드가 나타납니다.


---

### 💡 주피터 노트북 바로가기 아이콘 만드는 방법 (팁)

명령어를 입력할 필요 없이 클릭 한 번으로 실행하는 방법입니다.

1. 바탕화면 빈 곳에 마우스 오른쪽 클릭 -> **[새로 만들기]** -> **[바로 가기]**.
2. 항목 위치 입력창에 아래 내용을 그대로 복사해서 넣으세요.

   `cmd /c "jupyter notebook"`

   *(설명: cmd 창을 열어 주피터 노트북 명령을 실행하고 유지하라는 뜻입니다.)*

3. 바로 가기 이름은 **"Jupyter Notebook"**으로 지정하고 마칩니다.
4. 이제 이 아이콘을 더블클릭하면 바로 주피터 노트북이 실행됩니다.
