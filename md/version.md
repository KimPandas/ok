# 📊 파이썬 데이터 분석 패키지 버전

데이터 분석 환경 구축 시 가장 권장되는 안정적인 버전 조합입니다.

---

## 1. 프로그래밍 언어 (Runtime)
### **Python 3.11.9**
* **특징**: 최신 버전(3.12, 3.13)보다 라이브러리 호환성이 훨씬 뛰어납니다.
* **장점**: 회사 업무에 쓰이는 대부분의 패키지와 충돌 없이 작동하는 가장 성숙한 버전입니다.

👉  [파이썬 설치 가이드 (Python 3.11.9)](#파이썬-설치-가이드python-3119)

## 2. 개발 및 분석 환경 (IDE)
### **Jupyter Notebook 7.1.3**
* **설치**: 아나콘다(Anaconda) 없이 `pip` 명령어로 직접 설치 가능한 최신 노트북 버전입니다.
* **라이선스**: 기업에서도 라이선스 제약 없이 자유롭게 사용할 수 있습니다.

## 3. 데이터 분석 및 머신러닝 라이브러리

| 라이브러리명 | 버전 | 용도 |
| :--- | :--- | :--- |
| **NumPy** | `1.26.4` | 고성능 수치 계산 및 행렬 연산의 표준 |
| **Pandas** | `2.1.4` | 데이터프레임 기반의 데이터 전처리 및 분석 핵심 도구 |
| **Matplotlib** | `3.8.2` | 파이썬 시각화의 기초가 되는 그래프 드로잉 라이브러리 |
| **Seaborn** | `0.13.1` | 통계 차트를 더 아름답고 간편하게 그려주는 도구 |
| **Scikit-learn** | `1.4.0` | 머신러닝 모델 구축 및 데이터 전처리를 위한 필수 라이브러리 |
| **openpyxl / xlrd** | `최신` | 엑셀 파일(.xlsx, .xls)을 읽고 쓰기 위한 엔진 |
